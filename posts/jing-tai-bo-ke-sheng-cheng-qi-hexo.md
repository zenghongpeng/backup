今天我们来看看Hexo是什么东西！！

## 什么是Hexo
官方说Hexo是一个快速、简洁且高效的博客框架，使用Markdown引擎解析文章，几秒内就可以生成一个静态网站。

OK，官方都这么说了，那咱试试。

## 安装

安装Hexo前提要安装好`Node.js`、`Git`，需要安装的同学请参考各官方的或者在百度上搜索哈，我想既然想用Hexo搭建网站，这些知识还是必要的。

假设这些大家都有了，那咱么继续。

```shell
$ npm install -g hexo-cli
$ npm install hexo
```
